개인정보처리방침 / Privacy Policy

MaxiEdit (com.leegemma.maxiedit)

Effective: 2026-04-30 · Last updated: 2026-04-30

요약 / Summary

MaxiEdit는 어떤 개인정보도 수집·저장·전송하지 않습니다. 사용자가 선택한 사진과 영상은 디바이스 내부에서만 처리되며, 외부 서버로 전송되지 않습니다.

MaxiEdit does not collect, store, or transmit any personal data. Photos and videos you select are processed entirely on your device and are never uploaded to any server.

수집하는 데이터 / Data we collect

없음 / None

이 앱은 사용자의 개인정보를 수집하지 않으며, 분석(analytics)·광고·크래시 리포트 어떤 형태의 텔레메트리도 작동하지 않습니다.

This app collects no personal information. It does not run analytics, advertising, or any form of telemetry, including crash reports.

사용하는 권한 / Permissions used

사진·영상 라이브러리 접근 / Photo & video library access

편집할 사진/영상을 선택하기 위한 용도로만 사용됩니다. 선택한 미디어는 디바이스 메모리에서만 처리되며, 앱이 종료되면 사라집니다.

Used solely so you can pick the photos and videos you want to edit. Selected media stays in device memory only and is discarded when the app exits.

저장 공간 (선택) / Storage (optional)

완성한 콜라주를 사진/영상 파일로 디바이스에 저장하거나 시스템 공유 시트로 내보낼 때만 사용됩니다.

Used only when you save the finished collage as a photo or video file to your device, or share it via the system share sheet.

처리 위치 / Where processing happens

모든 편집·내보내기(이미지 합성, MP4 인코딩 등) 작업은 디바이스의 브라우저(WebView) 안에서 실행되며, 어떤 단계에서도 네트워크로 전송되지 않습니다.

All editing and export work (image composition, MP4 encoding, etc.) runs inside the device's browser/WebView. No step transmits anything over the network.

외부 자원 / Third-party resources

앱 UI를 표시하기 위해 다음 외부 자원을 처음 실행 시 다운로드합니다. 이 요청들은 일반적인 정적 자원 로드이며 사용자 식별 정보를 포함하지 않습니다.

  • Google Fonts (Inter, Noto Sans KR, Nanum Myeongjo)
  • jsDelivr CDN — Sortable.js, html2canvas, ffmpeg.wasm (지연 로드)

The app loads the following static assets at launch: Google Fonts and the jsDelivr CDN entries above. These are normal static-resource fetches and do not include any user identifier.

전체 라이선스: licenses.html

아동 데이터 / Children's data

이 앱은 어떤 데이터도 수집하지 않으므로 만 14세 미만 아동의 정보가 별도로 수집·처리되는 일이 없습니다.

Since the app collects no data at all, no information from children under 14 is collected or processed separately.

변경 사항 / Changes to this policy

방침이 갱신되면 이 페이지의 "Last updated" 날짜와 GitHub 커밋 히스토리에 반영됩니다. 별도 통지는 제공하지 않습니다.

If this policy changes, the "Last updated" date on this page and the GitHub commit history will reflect the change. Separate notifications are not provided.
